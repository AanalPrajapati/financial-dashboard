#  Smart Financial Insights Dashboard

## 📌 Overview
This project is a financial dashboard that helps users track their income, expenses, and overall balance. It also provides visual insights using charts.

---

## Features

- Dashboard summary (Balance, Income, Expense)
- Time-based chart (Line chart)
- Category-based chart (Pie chart)
- Transactions table
- Search functionality
- Role-based UI (Admin / Viewer)
- Insights (Highest spending category & total transactions)
- LocalStorage support

---

## Roles

- Viewer → Can view and search data
- Admin → Can add & delete and manage transactions

---

##  Tech Used

- HTML
- CSS
- JavaScript
- Chart.js

---

##  How to Run

1. Download or clone the project
2. Open index.html in your browser

---

## Insights

- Displays highest spending category
- Shows total transactions
- Visual representation using charts

---

##  Responsive

Works on both mobile and desktop screens
